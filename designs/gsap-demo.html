GSAP 核心动画示例

gsap.to / gsap.from / gsap.fromTo / stagger / ease / 控制

1. gsap.to() — 从当前位置动画到目标值

gsap.to(".box-1", { x: 200, rotation: 360, duration: 1, ease: "power2.out" })
to

2. gsap.from() — 从指定值动画到当前位置（入场）

gsap.from(".box-2", { y: -40, opacity: 0, scale: 0.5, duration: 0.8, ease: "back.out(1.7)" })
from

3. gsap.fromTo() — 显式指定起止值

gsap.fromTo(".box-3", { x: -100, opacity: 0 }, { x: 100, opacity: 1, duration: 1, ease: "power3.inOut" })
fromTo

4. Stagger — 依次错开动画

gsap.to(".box-4", { y: -60, scale: 1.2, duration: 0.5, stagger: 0.08, ease: "back.out(1.7)", yoyo: true, repeat: 1 })

7. autoAlpha — 淡入/淡出并自动管理 visibility

gsap.to(".box-7", { autoAlpha: 0, duration: 0.5 })
