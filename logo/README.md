# Patty Logo

## Files

| File | Format | Description |
|------|--------|-------------|
| patty-icon.svg | SVG | Vector source file |
| patty-icon.ico | ICO | Windows app icon (32x32) |

## Usage

The SVG file can be used for documentation and web contexts:

```markdown
![Patty Logo](logo/patty-icon.svg)
```

The ICO file is used as the Windows application icon and is located at:
```
resources/icon.ico
```
